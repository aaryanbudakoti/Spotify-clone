# 🎵 Spotify Clone

A simple **Spotify-inspired music player UI** built using **HTML & CSS**, with images and icons to replicate the look and feel of the original platform.  
<br><br>

## ✨ Features
- Clean and responsive layout  
- Sidebar navigation with playlists  
- Music player controls (play, pause, next, previous)  
- Styled using pure CSS and custom images  

## 🚀 Tech Stack
- HTML  
- CSS  
- Images/Icons  

This project focuses on **frontend design and styling**, showcasing how to recreate a modern music app interface with just HTML and CSS.
